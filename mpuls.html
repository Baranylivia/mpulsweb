About the MPULS Code

MPULS is a modular, multidimensional, parallel, well-documented numerical hydrodynamical code designed to model the envelope pulsation of classical pulsating variable stars.

We intend to make the code publicly available and introduce it in higher education courses. We will use the new code to better understand the stellar atmosphere, pulsation physics and related nonlinear phenomena.
